Paint Ball

Paintballs event comes under fun game in which all the players try to eliminate opponents by shooting each other with balls of coloured dye. The paintballs are shot through a device called as paintgun. When they hit a player, the dye marks that player as eliminated. The game is often played as an organized competitive game. In this fun game the field is scattered with some obstacles(natural or artificial) which players used to hide behind. This game is like a fun war game so it is played by wearing proper type of 'Battledress' (including goggles, facemasks,etc...) for safety purpose. As it game of quick thinking and decisively which makes you a star in it (just like a game of chess). Paintball is a character building game in which players learn about teamwork, gain self confidence and develop leadership abilities while having fun and getting welcome stress relief.

In Paintball fun war game there will be some amount of entry fees for playes and it will be organise on both days of 'PRAJWALAN' and as it is a type of fun game so there will not be any prize for the winners it will be organize just for a fun purpose.

Rules

  1. Each participant will be given 20 balls.
  2. Each participant will be given a 20-minute slot.
  3. If a participant wants an extra ball, then he/she will have to pay extra charges.
  4. The charge of extra one ball is 10 rs. i.e. 10rs/ball.

Terms and Conditions

If there is any kind of damage to any equipment, then the participant will be responsible for it and will compensate for it.
